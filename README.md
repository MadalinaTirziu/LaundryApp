# LaundryApp

A Windows Forms C# application for managing a laundry and dry cleaning business.

## Features
- Role-based login (Admin, Angajat, Client)
- Order management and tracking
- Service management (add, edit, delete)
- Reports generation
- System settings with backup
- Client profile and order history

## Tech Stack
- C# Windows Forms
- Entity Framework Core
- SQLite

## Roles
- **Admin** — manages users, services, settings and reports
- **Angajat** — processes and updates order statuses
- **Client** — places orders and tracks their status
